# Privacy Policy for Periodty

**Last updated:** 2026-01-01

Periodty ("the App") is a personal health tracking application designed to help users track menstrual cycles and related information. Your privacy is taken seriously. This Privacy Policy explains what data is collected, how it is used, and how it is protected.

---

## 1. Information We Collect

### 1.1 Personal and Health Data

Periodty allows users to optionally enter personal and health-related information, including but not limited to:

- Menstrual cycle dates  
- Period length and cycle length  
- User preferences related to reminders and tracking  
- Optional profile information (e.g., name)

This data is entered voluntarily by the user and may include **sensitive health data**.

### 1.2 Automatically Collected Information

Periodty does **not** automatically collect:

- Device identifiers  
- Location data  
- Advertising identifiers  
- Contacts  
- Media files  
- Analytics or usage data sent to external servers  

---

## 2. How Data Is Used

The data entered into Periodty is used **only within the app** to:

- Display cycle insights and summaries  
- Generate reports (PDF and CSV)  
- Show reminders and predictions locally on the device  
- Improve the user experience inside the app  

The data is **not used for advertising, tracking, profiling, or analytics**.

---

## 3. Data Storage and Security

- All data is stored **locally on the user’s device**
- Periodty does **not upload data to any external server** by default
- No cloud synchronization is enabled unless explicitly introduced in a future version and clearly disclosed to users

Reasonable technical measures are applied to protect locally stored data from unauthorized access.

---

## 4. Data Sharing

Periodty does **not**:

- Sell user data  
- Share data with third parties  
- Transfer data to external services  

Data may leave the device **only when the user explicitly exports or shares it**, such as generating a PDF or CSV report using the device’s sharing features.

---

## 5. User Control and Data Deletion

Users have full control over their data:

- All data can be edited or deleted within the app  
- Uninstalling the app permanently deletes all locally stored data  
- Exported files remain entirely under the user’s control  

---

## 6. Permissions

Periodty requests only the minimum permissions required for core functionality, such as:

- File access for exporting reports  
- Notifications for reminders (optional)  

Permissions can be revoked at any time through the device settings.

---

## 7. Children’s Privacy

Periodty is **not intended for children under the age of 13**.  
The app does not knowingly collect personal data from children.

---

## 8. Changes to This Privacy Policy

This Privacy Policy may be updated from time to time to reflect changes in features or legal requirements. Any changes will be communicated through the app or the app listing.
